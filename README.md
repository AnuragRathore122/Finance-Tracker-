# 💰 Django Finance Tracker

A full-stack personal finance tracking web application built with **Django** that helps users manage their income, expenses, and financial goals in one place. The application provides an intuitive dashboard to monitor spending habits, categorize transactions, and gain insights into personal finances.

## 🚀 Features

* 🔐 User authentication (Sign Up, Login, Logout)
* 💵 Add, edit, and delete income & expense transactions
* 📂 Categorize transactions for better organization
* 📊 Dashboard with financial summary
* 🎯 Set and track financial goals
* 📅 View transaction history
* 📱 Responsive and clean user interface
* 🗄️ Secure data storage using Django ORM

## 🛠️ Tech Stack

* **Backend:** Django, Python
* **Frontend:** HTML, CSS, JavaScript, Bootstrap/Tailwind CSS
* **Database:** SQLite (can be configured for PostgreSQL/MySQL)
* **Authentication:** Django Authentication System

## 🎯 Purpose

This project was built to strengthen my understanding of Django development, authentication, CRUD operations, database management, and full-stack web application development while creating a practical solution for personal finance management.
